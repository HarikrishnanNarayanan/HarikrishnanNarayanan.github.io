Project 3

Text Entailment and Semantic Relatedness

In this project, using machine learning and deep learning techniques 2 sentences were classified as: first sentence (the premise) entailing the second sentence (the hypothesis), contradicting it, or bearing no relation. The three classes are: ENTAILEMNT, CONTRADICTION, and NEUTRAL. An entailment relation exists if the premise sentence can cover the meaning of the hypothesis sentence. The dataset used was the SICK dataset which consists of 10,000 pairs of sentences annotated for semantic relatedness and entailment. The SICK data set was built starting from two existing paraphrase sets: the 8,000 ImageFlickr data set and the SEMEVAL-2012 Semantic Textual Similarity Video Descriptions dataset.
